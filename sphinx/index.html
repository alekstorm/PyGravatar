class pygravatar.User(email, password=None, apikey=None)¶

Represents a user account.

addresses()¶
Returns:a dictionary where each key is an email address belonging to the user account and each value is the Userimage:class assigned to it, or None if no image is assigned
deleteUserimage(id)¶
Delete the image from the user account, and unassign it from any email addresses.
exists(*addresses)¶
Return a dictionary where each key is an email address from the passed-in list and each value is a boolean of whether that email address belongs to a Gravatar account and has an image assigned to it. :param addresses: vararg list of email addresses to check
removeImage(*addresses)¶
For every email address passed in, unassign its image. :param addresses: vararg list of email addresses to be unassigned
saveData(data, rating)¶
Save the data as a new image in the user account. :param data: binary image data to save :param rating: integer rating for the new image (see Rating:class) :returns: ID of new image
saveUrl(url, rating)¶
Read the image pointed to by the URL and save it as a new image in the user account. :param url: URL pointing to an image to save :param rating: integer rating for the new image (see Rating:class) :returns: ID of new image
test()¶
Returns:the server’s number of seconds since the current epoch.
useUserimage(id, *addresses)¶
Assign the image identified by an ID to every email address passed in. :param id: image ID to assign :param addresses: vararg list of email addresses
userimages()¶
Returns:a list of Userimage:class objects belonging to the user account
