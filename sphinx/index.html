deleteImage(id)¶

Delete the image from the user account, and unassign it from any email addresses.

Parameter:id – ID of image to delete
emails()¶
Returns:a dictionary where each key is an email address belonging to the user account and each value is the Image:class assigned to it, or None if no image is assigned
exists(*emails)¶

Return a dictionary where each key is an email address from the passed-in list and each value is a boolean of whether that email address belongs to a Gravatar account and has an image assigned to it.

Parameter:emails – vararg list of email addresses to check
images()¶
Returns:a list of Image:class objects belonging to the user account
removeImage(*emails)¶

For every email address passed in, unassign its image.

Parameter:emails – vararg list of email addresses to be unassigned
